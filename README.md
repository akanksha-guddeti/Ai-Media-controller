🤖 AI Media Controller using Hand Gestures

An AI-powered media controller that enables users to control system volume using real-time hand gesture recognition through a webcam. The project leverages Computer Vision and Machine Learning to provide a touch-free and intuitive user experience.

✨ Features
🎥 Real-time hand tracking using webcam
✋ Hand gesture recognition
🔊 Control system volume with finger distance
⚡ Smooth and responsive gesture detection
🖥️ User-friendly interface
🤖 AI-based computer vision implementation
🛠️ Technologies Used
Python
OpenCV
MediaPipe
NumPy
Pycaw (Windows Volume Control)
Math Module
🚀 How It Works
Launch the application.
Allow webcam access when prompted.
Show your hand in front of the camera.
The application detects your hand landmarks.
Adjust the distance between your thumb and index finger to increase or decrease system volume.
The volume changes in real time based on the detected gesture.

Note: This is a desktop Python application and cannot run directly on GitHub.
